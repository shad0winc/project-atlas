# Recyclarr Configuration

This directory contains the version-controlled Recyclarr configuration for Project Atlas.

## Purpose

Recyclarr synchronizes quality profiles and custom formats from the TRaSH Guides into Sonarr and Radarr.

## Principles

- Configuration as Code
- One Source of Truth
- Version Controlled
- Repeatable Deployments

## Secrets

API keys are never stored here.

They are provided through environment variables.
