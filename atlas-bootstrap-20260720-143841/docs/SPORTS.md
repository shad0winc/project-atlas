# Project Atlas Sports Module

The Sports Module is an optional Project Atlas module for sports-focused media and live sports workflows.

## Status

Current status: Planned / Module Shell Created

## Design Goals

- Optional
- Isolated from the core media platform
- Compatible with Jellyfin
- Separately documented
- Separately configurable
- Safe to disable or remove

## Planned Components

- Sportyfin Docker service
- Sports media storage
- Jellyfin sports integration
- Team favorites
- League favorites
- Sports requests
- Sports dashboard
