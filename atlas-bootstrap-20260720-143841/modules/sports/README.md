# Atlas Sports Module

The Sports Module is an optional Project Atlas module for sports-focused workflows.

## Status

Module template created.

No production sports services are deployed yet.

## Purpose

This module will eventually provide:

- Sportyfin integration
- Sports-specific media storage
- Jellyfin integration
- Team favorites
- League favorites
- Sports requests
- Sports dashboard
